# shoes-project — брифинг для Claude

Веб-приложение для подбора кроссовок по удобству. Этот файл читается автоматически
в начале сессии. **Источник правды по плану и статусу — [docs/ROADMAP.md](docs/ROADMAP.md). Прочитай его первым.**

## Что это за продукт (честно)
**Примерка кроссовок, не подбор.** Пользователь видит, как модель **сидит на его
стопе**, через **рентген-оверлей**: полупрозрачный силуэт стопы под контуром
внутреннего пространства кроссовка — видно где тесно/свободно.
Поток на экране: **1) рисунок (оверлей) → 2) фидбэк словами** (подходит/тесно/свободно).
Оверлей строится из чисел (замер стопы + размер + репутация ширины), **без
генеративного AI** — приблизительно, 2D, с пометкой. Контур модели приблизительный
(колодки производителей закрыты). Реалистичное фото «как выглядит» — опция на потом.
Главный приоритет — **минимум галлюцинаций**: правда в данных и правилах. Без
ориентира (A4 или размера) честно говорим «недостоверно».

## Стек и конвенции
- Фронт: **Vite + React + TypeScript** в корне (`src/`), тесты — **Vitest**.
- Сервер: отдельная папка **`server/`**, чистый **`node:http`** (без Express),
  файлы **`.mjs`**, валидация — **zod**, тесты — встроенный **`node:test`**.
- Сервер по умолчанию на **порту 8788** (8787 занят соседним проектом wc26).
- LLM через **интерфейс провайдера** (`server/provider.mjs`): сейчас `mock`
  (фикстуры), потом `anthropic` (Claude API + веб-поиск), выбор через env `LLM_PROVIDER`.
  **Ключ API только на сервере**, никогда во фронте.
- Вердикт о посадке (шаг 3) будет на **строгих правилах**, LLM — только текст объяснения.
- Антигаллюцинации: структурированная схема, поле `confidence`, значения `unknown`
  вместо выдуманных чисел, подтверждение пользователем.

## Как запускать (из корня shoes-project)
```
npm install                   # фронт (один раз)
npm --prefix server install   # сервер (один раз)
npm --prefix server start     # API на :8788
npm run dev                   # фронт на :5173
npm test                      # тесты фронта
npm --prefix server test      # тесты сервера
```
